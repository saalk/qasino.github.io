## Folder structure
| folder           | what                                                   |
|:-----------------|:-------------------------------------------------------|
| /bin or /usr/bin | binaries or apps                                       |
| /etc             | config files                                           |
| /home/bob        | user folder                                            |
| /opt             | optional app packages                                  |
| /root            | home dir of root user (not /usr/root)                  |
| /srv             | websites                                               |
| /var             | writable counterpart to the /usr directory eg /var/log |
